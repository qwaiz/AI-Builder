# AI-Builder
AI Builder 是一个人工智能算法模型自动训练与生成工具。
AI Builder操作简单无需算法基础。只需将标注的数据导入到该工具，AI算法一键训练，一键导出，快速集成。无需算法基础，无需人干预。无需AI算法基础，普通人员就能依据需求生成AI算法。
